- Just start the Pong_2Players_1Keyboard_REFACTORED.html
- When the browser page opens with the game in it, the game starts automatically; there is no score limit
- You need to click on the game screen to hear the audio
- For a restart, just reload the page

Controls:
Player 1: left and right arrows for moving; down arrow for paddle speed increase
Player 2: A and D keys for moving; S key for paddle speed increase
